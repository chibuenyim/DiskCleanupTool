# 🧹 Disk Cleanup Tool v4.0 - Windows

## 🚀 Quick Start (It's Easy!)

### Step 1: Open the Tool
**Double-click `START.bat`**

That's it! The GUI will open automatically.

---

## 🎯 What You'll See

A simple window with:
- ✅ Checkboxes for what to clean
- 📊 How much space you'll free
- 🚀 Start button to begin
- 📤 Save/Load your settings

---

## 🎨 Two GUI Options

### Option 1: Standard GUI (Recommended for Everyone)
- All cleanup options
- Simple checkboxes
- Progress bar
- Results display

### Option 2: Enhanced GUI (Power Users)
- **Quick Presets:**
  - ⚡ Quick (1-2 GB in 5 min)

  - 💪 Deep (20-35 GB in 30 min)
- **Quick Actions:**
  - 🗑️ Empty Recycle Bin
  - 🌐 Clean Browser Cache
  - 👨‍💻 Clean Dev Cache
  - 🔄 Clean Update Cache
- Statistics tracking
- Export/Import settings

## 📋 What Can You Clean?

| Option | What It Cleans | Space Saved |
|--------|---------------|-------------|
| ✅ Windows Temp | Temporary files | 500 MB - 2 GB |
| ✅ Browser Caches | Chrome, Edge, Firefox | 300 MB - 1.5 GB |
| ✅ Developer Caches | npm, pip, Docker, etc. | 8-20 GB |
| ✅ App Caches | Spotify, Discord, Adobe | 1-5 GB |
| ✅ System Files | Logs, thumbnails, etc. | 3-12 GB |
| ✅ Windows.old | Old Windows installation | 10-30 GB |
| ✅ Hibernation | hiberfil.sys file | 2-8 GB |
| ✅ Windows Update | Update leftovers | 3-9 GB |

## 🎮 How to Use

1. **Double-click `START.bat`**
2. **Choose your GUI** (Standard or Enhanced)
3. **Check what you want to clean** (or use presets)
4. **Click "Start Cleanup"**
5. **Wait** (progress bar shows status)
6. **Done!** View results

---

## ⚙️ Advanced Options

### Want to Schedule Automatic Cleanup?
Choose "Scheduled Cleanup" from the menu → Runs every Sunday at 2 AM

### Want to Preview First?
Check "Dry Run" → Shows what would be cleaned without doing it

### Want to Save Your Settings?
Click "Export Settings" → Save your preferences
Next time: Click "Import Settings"

---

## 🆘 Need Help?

### Tool Won't Start?
- Right-click `START.bat` → "Run as Administrator"

### Want Command-Line Instead?
```batch
DiskCleanupTool.bat --All
```

### Want to See All Options?
```batch
DiskCleanupTool.bat --Help
```
